Welcome to the API documentation for the RESTful JSON API version of my professional resume.

Purpose

This API was written as a demonstration of my experience as a developer, using my skills in backend web development and devops.

  • This API is written in Rust, using Poem OpenAPI.
  • The data storage for this server is SQLite, using SQLx for querying and migrations.
  • The webserver is hosted via Docker on my personal homelab server.
  • The server is configured for Continuous Delivery.
    • Upon a push to the main branch, the webserver will rebuild itself automatically.
    • Every 15 minutes, crontab runs git pull.
    • A post-merge git hook executes make docker-reup when a change is pulled.
